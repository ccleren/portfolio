## 📌 Proyectos destacados

### 🏢 AWS Organizations Landing Zone

Arquitectura multi-cuenta empresarial con AWS Organizations, OUs, logs centralizados, Service Control Policies y AWS Budgets.

**Tecnologías:** AWS Organizations, SCP, IAM Identity Center, CloudTrail, AWS Budgets, Terraform.

---

### 🔁 Pipeline Jenkins para aplicación PHP Dockerizada

Aplicación PHP desplegada con Docker Compose, Apache, MySQL y phpMyAdmin, automatizada mediante un pipeline de Jenkins.

**Tecnologías:** Jenkins, Docker, Docker Compose, PHP, MySQL, Apache.

---

### ⚖️ Balanceo web con HAProxy y Apache en Docker

Escenario contenerizado con HAProxy como proxy inverso y varios servidores Apache como backends.

**Tecnologías:** Docker, Docker Compose, HAProxy, Apache.

---

### 📱 Reserva de Salas con Flutter y Firebase

Aplicación móvil para gestionar reservas de salas usando Flutter y Firebase Firestore.

**Tecnologías:** Flutter, Dart, Firebase, Firestore.
